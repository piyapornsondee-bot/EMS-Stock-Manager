backup สำรองข้อมูลอัตโนมัติไปยัง Google Sheets

ระบบสามารถสำรองประวัติการทำรายการรับเข้าและจ่ายออกของพัสดุ (Transactions) ไปยัง Google Sheets ได้โดยอัตโนมัติเมื่ออุปกรณ์เชื่อมต่ออินเทอร์เน็ต ผ่าน Google Apps Script Web App

link
สามารถนำ URL ที่ได้จากการ Deploy Apps Script มาวางตรงนี้
palette การแสดงผลและรูปลักษณ์ (Theme)
โหมดกลางคืน (Dark Theme)
เปลี่ยนสีพื้นหลังเพื่อถนอมสายตาในที่มืด
แจ้งเตือนเมื่อพัสดุมีสต๊อกต่ำกว่ากำหนด
แจ้งเตือนผ่านหน้าจอเมื่อพัสดุกู้ชีพต่ำกว่าเกณฑ์
code ซอร์สโค้ดสำหรับ Google Apps Script

1. สร้าง Google Sheet ใหม่
2. ไปที่เมนู Extensions > Apps Script
3. คัดลอกซอร์สโค้ดด้านล่างนี้ไปวางและบันทึก จากนั้นกด **Deploy > New deployment** (เลือกประเภท Web App, และตั้งค่า Who has access เป็น **Anyone**)
4. นำ URL ที่ได้มาตั้งค่าด้านบน

database การจัดการฐานข้อมูลเครื่อง (Local Database)

ข้อมูลทั้งหมดบันทึกอยู่ภายในบราวเซอร์ของคุณ (IndexedDB) คุณสามารถสำรองข้อมูล ล้างข้อมูล หรือคืนค่าเริ่มต้นระบบได้ที่นี่
