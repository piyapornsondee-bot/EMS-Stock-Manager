EMS Stock Manager
User Manual

คู่มือการใช้งานระบบบริหารจัดการคลังพัสดุการแพทย์สำหรับหน่วย EMS และโรงพยาบาล ครอบคลุมทุกฟีเจอร์และขั้นตอนการใช้งาน

format_list_bulleted สารบัญ
login การเข้าสู่ระบบ dashboard Dashboard inventory_2 คลังพัสดุ add_shopping_cart รับพัสดุเข้า shopping_cart_checkout จ่ายพัสดุออก qr_code_scanner การสแกน QR receipt_long ประวัติการทำรายการ bar_chart รายงาน manage_accounts จัดการผู้ใช้ settings ตั้งค่า shield สิทธิ์การใช้งาน offline_bolt การใช้งานออฟไลน์ code เครดิตผู้พัฒนา
login
1. การเข้าสู่ระบบ
วิธีเข้าใช้งานระบบ EMS Stock Manager
1
เปิดระบบ
เปิด Browser และเข้าสู่ URL ของระบบ EMS Stock Manager ที่ได้รับจากผู้ดูแลระบบ หรือเปิดผ่าน App ที่ติดตั้งบนอุปกรณ์
2
กรอกชื่อผู้ใช้และรหัสผ่าน
กรอก ชื่อผู้ใช้ (Username) และ รหัสผ่าน (Password) ที่ได้รับจากผู้ดูแลระบบ
3
กดปุ่ม "เข้าสู่ระบบ"
กดปุ่ม เข้าสู่ระบบ เพื่อยืนยัน ระบบจะตรวจสอบข้อมูลและนำท่านไปยังหน้า Dashboard
4
ออกจากระบบ
กดที่ชื่อผู้ใช้ด้านล่าง Sidebar หรือกดปุ่ม logout เพื่อออกจากระบบ
warning หากลืมรหัสผ่าน กรุณาติดต่อผู้ดูแลระบบ (Admin) เพื่อรีเซ็ตรหัสผ่าน
lock รหัสผ่านถูกเข้ารหัส devices รองรับทุกอุปกรณ์ install_mobile ติดตั้งเป็น App ได้
dashboard
2. Dashboard — ภาพรวมสต๊อก
หน้าหลักที่แสดงสรุปสถานะคลังพัสดุแบบ Real-time

Dashboard แสดงข้อมูลสรุปภาพรวมของคลังพัสดุ ได้แก่:

📊
KPI Cards
แสดงจำนวนพัสดุทั้งหมด, สต๊อกรวม, พัสดุสต๊อกต่ำ, และรายการวันนี้
📈
กราฟรับ/จ่าย 7 วัน
แสดงแนวโน้มการรับเข้าและจ่ายออกพัสดุย้อนหลัง 7 วัน
⚠️
รายการสต๊อกต่ำ
แสดงรายการพัสดุที่ต้องเติมด่วน พร้อม Progress Bar บอกปริมาณที่เหลือ
🕐
รายการล่าสุด
แสดงประวัติการทำรายการ 6 รายการล่าสุดแบบ Real-time
info
กดที่ KPI Card เพื่อไปยังหน้าคลังพัสดุที่กรองข้อมูลตามหมวดหมู่นั้นๆ โดยอัตโนมัติ
lightbulb
หากมีพัสดุสต๊อกต่ำ จะมี Badge แจ้งเตือน (จุดแดง) ปรากฏที่เมนู คลังพัสดุ ใน Sidebar
inventory_2
3. คลังพัสดุ — Inventory
ดู, ค้นหา, และจัดการพัสดุทั้งหมดในระบบ
🔍
ค้นหาพัสดุ
พิมพ์ชื่อพัสดุ, บาร์โค้ด, หรือหมวดหมู่ในช่อง Search เพื่อกรองรายการ
📍
กรองตามสถานที่
เลือก Dropdown สถานที่เพื่อดูสต๊อกของแต่ละจุด
⚠️
ฟิลเตอร์สต๊อกต่ำ
กดปุ่ม "สต๊อกต่ำ" เพื่อดูเฉพาะพัสดุที่มีปริมาณต่ำกว่าค่าขั้นต่ำที่กำหนด
➕
เพิ่มพัสดุใหม่
กดปุ่ม "+ เพิ่มพัสดุ" กรอกข้อมูลชื่อ, หน่วย, สถานที่, จำนวนขั้นต่ำ และ Barcode
✏️
แก้ไข / ลบพัสดุ
กดปุ่ม edit เพื่อแก้ไข หรือ delete เพื่อลบ (เฉพาะ Admin เท่านั้น)
📋
สลับมุมมอง Grid / List
กดไอคอนมุมมองที่มุมขวาบนเพื่อสลับระหว่างการแสดงผลแบบ Grid และ List
lightbulb Stock Bar สีแดง = ต้องเติมด่วน, สีเหลือง = ใกล้หมด, สีเขียว = ปกติ
add_shopping_cart
4. รับพัสดุเข้า — Receive Stock
บันทึกการรับพัสดุเวชภัณฑ์เข้าคลัง
1
เลือกสถานที่
เลือกสถานที่ที่จะรับพัสดุเข้า
2
ค้นหาหรือสแกนพัสดุ
พิมพ์ชื่อพัสดุในช่องค้นหา หรือกดปุ่ม qr_code_scanner เพื่อสแกน QR/Barcode
3
เพิ่มลงตะกร้า
กด "+ เพิ่ม" เพื่อเพิ่มพัสดุลงในรายการรับเข้า ปรับจำนวนได้ด้วยปุ่ม +/-
4
ยืนยันการรับเข้า
ตรวจสอบรายการในตะกร้า จากนั้นกดปุ่ม "ยืนยันรับเข้า" ระบบจะอัปเดตสต๊อกทันที
info ระบบจะบันทึก Log ผู้ที่รับพัสดุเข้า, วันเวลา, และจำนวนไว้ในประวัติการทำรายการอัตโนมัติ
shopping_cart_checkout
5. จ่ายพัสดุออก — Issue Stock
บันทึกการเบิกจ่ายพัสดุออกจากคลัง
1
เลือกสถานที่
เลือกสถานที่ที่จะเบิกพัสดุออก เพื่อให้ระบบดึงสต๊อกจากจุดนั้นโดยอัตโนมัติ
2
ค้นหาหรือสแกนพัสดุ
พิมพ์ชื่อพัสดุหรือสแกน QR Code เพื่อเพิ่มลงรายการจ่ายออก
3
ระบุจำนวนที่ต้องการเบิก
กรอกจำนวนที่ต้องการ ระบบจะแสดงเตือนหากจำนวนเกินสต๊อกที่มีอยู่
4
ยืนยันการจ่ายออก
กดปุ่ม "ยืนยันจ่ายออก" ระบบจะหักสต๊อกและบันทึกรายการทันที
error ไม่สามารถจ่ายพัสดุออกเกินจำนวนที่มีในสต๊อก หากสต๊อกเป็น 0 ปุ่มจ่ายออกจะถูก Disable อัตโนมัติ
qr_code_scanner
6. การสแกน QR Code / Barcode
ใช้กล้องอุปกรณ์สแกนพัสดุอย่างรวดเร็ว

ระบบรองรับการสแกน QR Code และ Barcode ผ่านกล้องของอุปกรณ์ เพื่อความรวดเร็วในการรับ/จ่ายพัสดุ

qr_code_scanner
ขั้นที่ 1
กดปุ่ม สแกน QR ที่หน้า รับเข้า หรือ จ่ายออก
photo_camera
ขั้นที่ 2
อนุญาตการเข้าถึงกล้องเมื่อ Browser ถาม
center_focus_strong
ขั้นที่ 3
จ่อกล้องไปที่ QR Code หรือ Barcode ของพัสดุ
check_circle
ขั้นที่ 4
ระบบค้นหาพัสดุโดยอัตโนมัติและเพิ่มลงรายการทันที
lightbulb สามารถพิมพ์ QR Code ของพัสดุแต่ละรายการจากหน้า คลังพัสดุ เพื่อติดที่กล่องหรือตู้เก็บพัสดุ
receipt_long
7. ประวัติการทำรายการ — Transactions
ดูบันทึกการรับ/จ่ายพัสดุทั้งหมดในระบบ
📅
กรองตามวันที่
เลือกช่วงวันที่เริ่มต้น-สิ้นสุด เพื่อดูประวัติในช่วงเวลาที่ต้องการ
🔍
ค้นหาและกรอง
ค้นหาด้วยชื่อพัสดุ, ผู้ทำรายการ, ประเภทรายการ (รับเข้า/จ่ายออก)
📊
ดูรายละเอียด
แต่ละรายการแสดง ชื่อพัสดุ, จำนวน, สถานที่, ผู้ทำรายการ, และวันเวลา
info รายการทำงานแบบ Real-time — ข้อมูลจะอัปเดตอัตโนมัติเมื่อมีการทำรายการใหม่
bar_chart
8. รายงาน — Reports
วิเคราะห์และส่งออกข้อมูลการใช้พัสดุ
📈
รายงานการใช้งาน
ดูสถิติการรับเข้า-จ่ายออกพัสดุในแต่ละช่วงเวลา พร้อมกราฟแสดงผล
🏆
พัสดุใช้มากที่สุด
แสดง Top 10 พัสดุที่ถูกเบิกใช้บ่อยที่สุด สำหรับวางแผนจัดซื้อ
📤
Export ข้อมูล
ส่งออกรายงานเป็นไฟล์ PDF หรือ CSV เพื่อนำไปใช้งานต่อ
lightbulb
ใช้รายงานเพื่อวางแผนจัดซื้อพัสดุล่วงหน้า โดยดูจากอัตราการใช้งานเฉลี่ยต่อเดือน
manage_accounts
9. จัดการผู้ใช้ — User Management
เพิ่ม, แก้ไข, และจัดการสิทธิ์ผู้ใช้ระบบ (เฉพาะ Admin)
➕
เพิ่มผู้ใช้ใหม่
กรอกชื่อ-นามสกุล, Username, Password และกำหนด Role (Admin / Staff / Viewer)
✏️
แก้ไขข้อมูลผู้ใช้
เปลี่ยนชื่อ, รหัสผ่าน, หรือ Role ของผู้ใช้ได้ตลอดเวลา
🔒
ระงับ / เปิดใช้บัญชี
สามารถระงับบัญชีผู้ใช้ชั่วคราวโดยไม่ต้องลบออกจากระบบ
settings
10. ตั้งค่าระบบ — Settings
ตั้งค่าการสำรองข้อมูลและรูปแบบการแสดงผล
📤
Google Sheets Backup
ตั้งค่า URL ของ Google Apps Script เพื่อสำรองข้อมูล Transaction ไปยัง Google Sheets โดยอัตโนมัติ
🔄
Auto Sync
ระบบจะ Sync ข้อมูลอัตโนมัติทุกครั้งที่อุปกรณ์เชื่อมต่ออินเทอร์เน็ต
🗑️
รีเซ็ตข้อมูล
คืนค่าข้อมูลพัสดุและบัญชีผู้ใช้กลับสู่ค่าเริ่มต้น (เฉพาะ Admin เท่านั้น)
error การรีเซ็ตข้อมูลเป็นการกระทำที่ไม่สามารถย้อนกลับได้ กรุณาใช้ด้วยความระมัดระวัง
